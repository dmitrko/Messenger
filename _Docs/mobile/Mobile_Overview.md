# Мобильный Клиент (React Native)

## Обзор
Мобильная версия NostaChat реализована на базе **React Native**. Основная цель мобильного клиента — обеспечить такой же высокий уровень приватности, как и десктопная версия, сохраняя при этом удобство использования на смартфонах.

## Текущий статус
Разработка мобильного клиента находится на ранней стадии. Основной функционал на данный момент включает:
- Базовую структуру приложения на TypeScript.
- Подключение к WebSocket-серверу для регистрации и обмена сообщениями.
- Обертки для интеграции `crypto` библиотек.

## Планы на развитие
В ближайшее время планируется:
1. **Интеграция Libsodium**: Портирование и настройка `react-native-libsodium` для работы с ключами X25519.
2. **Kyber на нативах**: Реализация или интеграция модуля для постквантового шифрования Kyber-768 на мобильных устройствах.
3. **WebRTC для Android/iOS**: Настройка `react-native-webrtc` для поддержки P2P-соединений между десктопом и мобильным устройством.
4. **Push-уведомления**: Реализация системы уведомлений о новых сообщениях без ущерба приватности (например, использование зашифрованных Payload в уведомлениях).

## Технический стек
- **React Native** (v0.73+).
- **TypeScript**.
- **WebSocket** (через стандартный API браузера/React Native).
- **Tailwind CSS** (через `nativewind` или аналогичные библиотеки стилизации).
